# LeenBattleSimulator

Unity ECS-based battle simulator built using Unity 2022.3 LTS and Entities 1.4.4.  
This project demonstrates ECS architecture including targeting, movement, combat, and ScriptableObject-driven team configuration.

---

## Overview

This project was developed as part of a technical assessment to demonstrate the use of Unity's Entity Component System (ECS) for gameplay simulation.

The simulation features multiple teams of units that automatically acquire targets, move toward enemies, attack, and resolve combat until one team wins.

---

## Features

- ECS-based unit simulation
- Automatic target acquisition
- Movement toward enemy units
- Attack, damage, and death systems
- ScriptableObject-based team configuratio
